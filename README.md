# Taxi Ride Analysis Dashboard

## Project Overview
This project focuses on analyzing taxi ride data to uncover business insights related to ride demand, revenue, customer behavior, payment preferences, and operational performance.
The raw dataset was first cleaned and prepared in Microsoft Excel, followed by data modeling, DAX calculations, and interactive dashboard development in Power BI. The final dashboard helps identify performance trends and supports data-driven business decisions.

## Project Objective
To transform raw taxi ride data into actionable business insights by analyzing ride demand, revenue trends, customer behavior, and operational performance through interactive Power BI dashboards.

## Tools & Technologies
* Microsoft Excel
* Power BI
* DAX

## Dataset Information
* Dataset Type: Synthetic Taxi Ride Dataset
* Initial Total Records: ~9,000
* Clean Dataset Records used for Analysis: ~8600
* Total Columns: 14
* Ride Types: Rapido Bike, Uber Go, Uber Premier, etc.
* Cities Included: Delhi, Mumbai, Pune, Bangalore and others

## Data Cleaning & Preparation
The dataset contained intentionally introduced inconsistencies to simulate real-world business data.

Cleaning tasks performed:
* Handled missing values
* Removed spelling inconsistencies
* Standardized categorical values
* Checked duplicate Ride IDs
* Corrected invalid values
* Treated fare outliers
* Fixed inconsistent text formatting
* Validated ride duration and ride status
* Prepared clean dataset for analysis

-> Key Business Insights:
* Delhi generated the highest revenue (₹4.48L).
* Pune recorded the lowest revenue (₹3.89L).
* Rapido Bike was the most preferred service type (25.97% of rides).
* Mumbai recorded the highest number of completed rides.
* Delhi recorded the highest ride cancellations.
* Rapido Bike had the highest cancelled rides.
* UPI was the most preferred payment method (25.88%).
* High-demand services also experienced higher cancellation volumes.

 -> Business Recommendations:
* Maintain strong driver availability in high-performing cities.
* Review operational factors affecting revenue in Pune.
* Improve bike driver availability during peak hours.
* Analyze Mumbai's successful ride completion strategy.
* Investigate cancellation patterns in Delhi.
* Optimize Rapido Bike operations to reduce cancellations.
* Continue promoting digital payment adoption.
* Balance driver supply with ride demand to improve reliability.
